Phase 5.1 Email & Scheduled Notifications Setup

1) Keep .env local only. Do not push it to GitHub.

2) Add these values to your local .env:

MAIL_SERVER=smtp.gmail.com
MAIL_PORT=587
MAIL_USE_TLS=True
MAIL_USERNAME=your_email@gmail.com
MAIL_PASSWORD=your_google_app_password
MAIL_DEFAULT_SENDER=your_email@gmail.com

# Optional scheduler controls
ENABLE_EMAIL_SCHEDULER=False
EMAIL_SCHEDULER_INTERVAL_MINUTES=60
SEND_DAILY_SUMMARY_EMAIL=False

3) Gmail note:
Use a Google App Password, not your normal Gmail password.

4) Run this SQL file in SQL Server Management Studio:
sql/phase5_1_email_notifications.sql

5) Restart Flask:
taskkill /F /IM python.exe
python app.py

6) Open Dashboard and use the Phase 5.1 Email Notifications panel:
- Send test email
- Run deadline check
- Daily summary

7) Only after test email works, enable scheduler if you want automatic checking:
ENABLE_EMAIL_SCHEDULER=True

The scheduler checks due today, due tomorrow, and overdue tasks. Duplicate emails are prevented using email_notification_logs.unique_key.
