LifeOS AI — Phase 5.1 Professional Notifications Upgrade
========================================================

This upgrade turns the basic email feature into user-controlled smart notifications.

Added features
--------------
1. Notification Preferences page
   /notifications/settings

2. User can enable/disable:
   - All email notifications
   - Task deadline reminders
   - Custom task reminders
   - Overdue alerts
   - Daily checkup email
   - Weekly summary email
   - Monthly analytics email
   - Project deadline/risk alert switches for future project health integration

3. Custom task reminders
   - Each task can now have its own reminder date/time.
   - Reminders are sent only if the user's preferences allow them.

4. Daily Checkup email
   - Open tasks
   - Overdue tasks
   - Due today
   - Due in the next 7 days

5. Weekly Summary email
   - Completed/open/overdue tasks
   - Tasks due this week

6. Monthly Analytics email
   - Total tasks
   - Completed/open/overdue/blocked tasks
   - Completion rate
   - Active projects
   - Average project progress
   - Most active project

7. Notification History page
   /notifications/history

Database step
-------------
Run this SQL file in SQL Server Management Studio:

sql/phase5_1_professional_notifications.sql

Keep the previous email setup SQL too if you did not run it yet:

sql/phase5_1_email_notifications.sql

Environment variables
---------------------
Make sure your .env still has:

MAIL_SERVER=smtp.gmail.com
MAIL_PORT=587
MAIL_USE_TLS=True
MAIL_USERNAME=your_email@gmail.com
MAIL_PASSWORD=your_google_app_password
MAIL_DEFAULT_SENDER=your_email@gmail.com

For testing, keep:

ENABLE_EMAIL_SCHEDULER=False
EMAIL_SCHEDULER_INTERVAL_MINUTES=60

After manual buttons work, enable:

ENABLE_EMAIL_SCHEDULER=True

How to test
-----------
1. Restart Flask.
2. Open Notification Settings from the sidebar.
3. Send a test email.
4. Enable Daily Checkup and Monthly Analytics.
5. Press Run Smart Check.
6. Open Notification History to verify logs.
7. Add or edit a task and enable a custom reminder.
