LifeOS AI — Phase 5.5 Balanced Focus Studio
============================================

This build replaces the two previous extremes:
- the crowded Focus Studio v2.2 interface
- the overly empty Minimal Focus Mode interface

The new layout keeps the main flow clear while still providing useful tools.

WHAT CHANGED
------------
Preparation screen:
- Task selection with workspace, priority, and deadline preview
- Quick Sprint (25 min), Deep Work (50 min), Flow Block (90 min), and Custom duration
- User-selected break length: 5, 10, or 15 minutes
- Four calm workspace palettes: Mist, Sage, Lavender, and Sand
- Generated ambient audio: Rain, Brown noise, White noise, or None
- Optional fullscreen, completion chime, and keep-screen-awake controls
- Clear session summary before starting

Active session:
- Spacious circular timer
- Calm light background instead of the previous dark navy screen
- Pause, extend, and finish controls
- Break timer without ending the focus session
- Distraction inbox with saved thoughts
- Convert a parked thought into a general LifeOS task
- Theme, sound, volume, fullscreen, and wake-lock tools in a side drawer
- Completion choices: break, continue overtime, or review
- Timer recovery after refresh remains supported

INSTALLATION
------------
1. Stop Flask completely:

   taskkill /F /IM python.exe

2. Extract this ZIP directly into your existing LifeOS project folder.

3. Choose "Replace all files" when Windows asks.

4. No new SQL migration is required for this update.
   It uses the existing focus_sessions and focus_distractions tables.

5. Start LifeOS:

   python app.py

6. Open:

   http://127.0.0.1:5000/focus/

TEST CHECKLIST
--------------
1. Open Focus Studio and confirm the main area is light and calm.
2. Select a task and confirm its project, priority, and deadline update.
3. Test all four focus durations.
4. Choose a break length, palette, and ambient sound.
5. Start a session.
6. Test pause, +5 minutes, fullscreen, and focus tools.
7. Start and end a break.
8. Park a thought and convert it into a task.
9. Refresh during the session and confirm the timer recovers.
10. Finish the session and save the review.
