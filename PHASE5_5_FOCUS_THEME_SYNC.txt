LifeOS Phase 5.5 — Focus Theme Sync

Changes:
- Focus setup dashboard now uses the same LifeOS light/dark theme variables.
- Focus Insights now matches the rest of the application.
- Cards, forms, buttons, borders and shadows follow the global design system.
- Review modal and Focus side panels now adapt to light/dark mode.
- The immersive timer can still use Mist, Sage, Lavender or Sand as an optional session environment.

Installation:
1. Stop Flask.
2. Extract this ZIP into the LifeOS project folder and replace files.
3. Restart with: python app.py

No SQL migration is required.
