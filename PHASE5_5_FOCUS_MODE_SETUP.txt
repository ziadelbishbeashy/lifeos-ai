LifeOS AI — Phase 5.5 Focus Mode

1. Run in SQL Server Management Studio:
   sql/phase5_5_focus_mode.sql

2. Restart Flask:
   taskkill /F /IM python.exe
   python app.py

3. Open:
   http://127.0.0.1:5000/focus/

Included:
- Select a general or project task
- 25/50/90-minute presets and custom duration
- Live timer with pause/resume
- Distraction counter
- Finish-session review and notes
- Optional task completion
- Daily focus totals
- Recent focus-session history

Smart Daily Planner is intentionally skipped for now and not mixed into this phase.
